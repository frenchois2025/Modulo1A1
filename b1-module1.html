🔗 Les pronoms relatifs qui et que

📚 Rappel grammatical
PronomFonctionExemple
quiSujet du verbe qui suitC'est l'ami qui parle vite.
que / qu'Complément d'objet directLe film que j'ai vu était super.

💡 qui = sujeto  |  que = complemento directo. Ante vocal, que → qu'.

4. J'ai rencontré une collègue ___ travaille à Lyon.

Escribe el pronombre relativo correcto en cada espacio.

📍 Le pronom relatif où

📚 Rappel grammatical
PronomRemplaceExemple
où (lieu)Un complément de lieuLa ville où je suis né est belle.
où (temps)Un complément de tempsLe jour où tu es parti, il a plu.

💡 En français, où remplace aussi les expressions de temps : le jour, le moment, l'époque… (en español : "cuando" o "en que")

Ejercicio 1 · ¿Lugar o tiempo?

2. Je me souviens du soir ___ nous nous sommes rencontrés.
3. Le pays ___ elle vit est très chaud.
4. L'époque ___ mes parents étaient jeunes était différente.
🗺️
¡Bien localizado!
Ejercicio 2 · Completa con qui, que ou où
Elige el pronombre relativo correcto.
1. C'est la rue ___ j'ai grandi.
Pista: ¿Hace referencia a un lugar?
2. Le moment ___ j'ai su la vérité a changé ma vie.
Pista: Momento temporal → où.
3. Le café ___ nous allons souvent est près de la gare.
Pista: Lugar al que van → où.

🔑 Le pronom relatif dont

📚 Rappel grammatical
CasExplicationExemple
dont (verbe + de)Remplace de + nom après un verbeLe sujet dont il parle est complexe.
(parler de)
dont (adjectif + de)Remplace de + nom après un adjectifLe film dont je suis fier a gagné.
(être fier de)
dont (possession)Indique la possessionL'ami dont le père est médecin…

💡 Clave: si el verbo o adjectivo pide de, usa dont. Ej: avoir besoin de, parler de, être content de, se souvenir de…

Ejercicio 1 · Usar dont correctamente
Elige el pronombre relativo adecuado.
1. C'est le livre ___ j'ai besoin.
2. Voilà le projet ___ je suis vraiment fier.
3. C'est une histoire ___ tout le monde se souvient.
4. J'ai un ami ___ la mère est écrivaine.
🔑
¡Dominas el dont!
Ejercicio 2 · Completa con dont, qui, que ou où
Completa con el pronombre relativo correcto.
1. La personne ___ nous avons parlé n'est pas venue.
Pista: parler de → dont.
2. C'est la ville ___ je suis originaire.
Pista: être originaire de → dont.
3. Le film ___ tout le monde parle sort vendredi.
Pista: parler de → dont.
4. C'est une recette ___ les ingrédients sont simples.
Pista: Posesión (les ingrédients de la recette) → dont.
✍️
¡Muy bien!

⚙️ Les pronoms relatifs composés

📚 Rappel grammatical — lequel / laquelle / lesquels / lesquelles
FormeAvec àAvec de
lequel (m. sg.)auquelduquel
laquelle (f. sg.)à laquellede laquelle
lesquels (m. pl.)auxquelsdesquels
lesquelles (f. pl.)auxquellesdesquelles

💡 Se usan tras preposiciones (avec, pour, sur, dans, à, de…) cuando el antecedente es una cosa. Con personas, se puede usar qui : la personne avec qui je travaille.

Ejercicio 1 · Pronoms composés
Selecciona la forma correcta de lequel.
1. C'est le problème ___ je pensais.
2. Voilà la table sur ___ j'ai posé les clés.
3. Ce sont des résultats sur ___ nous pouvons compter.
4. L'entreprise pour ___ il travaille est internationale.
⚙️
¡Bravo !
Ejercicio 2 · Completa con la forma correcta
Elige entre auquel / à laquelle / auxquels / lequel / laquelle / lesquelles…
1. C'est le cours ___ je m'inscris demain.
Pista: s'inscrire à + masculino singular → auquel.
2. Voici les raisons pour ___ j'ai choisi ce métier.
Pista: les raisons → femenino plural → lesquelles.
3. Le bureau dans ___ il travaille est au 3e étage.
Pista: le bureau → masculino singular → lequel.
✍️
¡Perfecto!
