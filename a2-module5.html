📣 L'impératif présent

Ejercicio 1 · Formación del impératif
El imperativo no usa pronombre sujeto. Selecciona la forma correcta.
1. ___ tes devoirs ! (tu → faire)
2. ___ à vos places ! (vous → s'asseoir)
3. ___ ensemble ! (nous → chanter)
4. ___ ici, s'il te plaît. (tu → venir)
Ejercicio 2 · Conjuga en impératif
Escribe la forma imperativa del verbo indicado.

1. (écouter / tu) → bien le professeur !

2. (finir / vous) → vos exercices avant midi.

3. (être / nous) → ponctuels !

4. (prendre / tu) → ce médicament deux fois par jour.

Ejercicio 3 · Consejos de salud
Asocia el consejo en imperativo con la situación adecuada.
Ejercicio 4 · Imperativo negativo
El imperativo negativo = ne + verbo + pas. Transforma las frases.

1. Mange des sucreries → Ne pas de sucreries.

2. Courez dans le couloir → Ne pas dans le couloir.

3. Buvez de l'alcool → Ne pas d'alcool.

4. Oublie tes médicaments → N' pas tes médicaments.
